# Mayamin.github.io

🛠️ Built With
HTML5/CSS3: Structured with semantic HTML and styled using advanced CSS features like CSS Variables (for theme toggling), Flexbox, and Grid.

JavaScript (Vanilla): Used for dynamic theme switching, intersection observers for scroll animations, and interactive navigation.

Google Fonts: Utilizes Inter and Sora for a clean, professional aesthetic.

✨ Key Features
Dynamic Theme Toggling: Includes a built-in Dark/Light mode switch that respects user system preferences and saves them via localStorage.

Responsive Design: Mobile-first architecture ensures a seamless experience across desktops, tablets, and smartphones.

Interactive UI: Smooth scrolling, reveal-on-scroll animations, and hover-triggered shadow effects for project and publication cards.

Comprehensive Sections:

About: High-level bio and key professional stats.

Experience: Chronological timeline of research and professional roles.

Projects & Demos: Showcases high-impact work like the NASA-funded FIRETWIN project with dedicated media cards.

Publications: Organized list of academic papers with venue details and citation links.

Skills: Visualized proficiency bars for technical expertise.

Contact: Integrated contact portal and professional social links.
